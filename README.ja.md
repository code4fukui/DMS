# DMS

度、分、秒を10進度に変換します。

## 使い方
```js
import { dms2d, sdms2d } from "https://code4fukui.github.io/DMS/DMS.js";

console.log(dms2d(35, 10, 5));
console.log(sdms2d("35 10 5N"));
```

## 機能
- 度、分、秒を10進度に変換
- 文字列形式の座標のパースをサポート

## 要件
なし。モダンなJavaScript環境で動作します。
